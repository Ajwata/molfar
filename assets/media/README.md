# Медиа-файлы Мольфар

## Как заменить плейсхолдеры на реальные фото и видео

Просто положите файлы с правильными именами в эту папку — HTML автоматически их подхватит. Если файла нет — показывается SVG-плейсхолдер.

## Видео (Hero)

| Файл | Назначение | Формат | Рекомендации |
|------|-----------|--------|--------------|
| `hero-bg.mp4` | Фоновое видео в hero | H.264 MP4 | 1920×1080, 10-20 сек, до 8 МБ, без звука |
| `hero-bg.webm` | Альтернативный формат (опционально) | VP9 WebM | Тот же контент, лучше сжатие |
| `hero-poster.jpg` | Заглушка пока видео грузится | JPG | 1920×1080 |

**Сжать видео:** [Handbrake](https://handbrake.fr/) или ffmpeg:
```
ffmpeg -i input.mp4 -c:v libx264 -crf 28 -preset slow -an -vf scale=1920:1080 hero-bg.mp4
```

### Карточки инфраструктуры
Просто положите файлы — JS сам проверит их наличие и применит как фон карточки с тёмным оверлеем:
- `infra-spa.jpg` — SPA & басейн
- `infra-lake.jpg` — Озеро
- `infra-restaurant.jpg` — Ресторан
- `infra-activities.jpg` — Активності
- `infra-kids.jpg` — Дитяча зона
- `infra-security.jpg` — Безпека

Если файла нет — карточка остаётся с обычным тёмным фоном и иконкой. Если файл есть — он автоматически становится фоном с градиентом затемнения.

## Оптимизация

**Конвертация в WebP** (в 2-3 раза легче JPG):
```
cwebp -q 80 photo.jpg -o photo.webp
```
Или онлайн: [squoosh.app](https://squoosh.app)

Для каждой фотки можно положить и `.jpg` и `.webp` — браузер сам выберет лучший формат:
- Если есть `concept.webp` — браузер использует его
- Иначе берёт `concept.jpg`
- Иначе показывается `concept-placeholder.svg`
